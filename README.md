# Math - School Project

A complete frontend + backend math learning website for students.

## Features

- Colorful and responsive UI
- Math quiz gameplay (addition, subtraction, multiplication, division)
- Levels, score, streak, and lives system
- Hint button
- Daily goal tracker
- Leaderboard
- Progress save API (username, best score, level, streak)

## Tech

- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Node.js + Express
- Storage: local JSON file (`data/progress.json`)

## Run

```bash
npm install
npm start
```

Then open:

http://localhost:3000
